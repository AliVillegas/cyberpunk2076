## To run
install nodejs
and run the following at /cyberpunk2076/:

```
npm install
npm run
```
## Deployment

Bundle your code, and push it in your repo:
```bash
npm run build
git add
git commit -m "Deploying on GitHub Pages"
git push
```
## Formatting
The code can be formated with prettier:
```bash
npm run format
```
